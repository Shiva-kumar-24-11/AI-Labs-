# 🤖 Artificial Intelligence Lab (01CT0616)

This repository contains all Artificial Intelligence lab experiments performed as part of my **B.Tech ICT curriculum at Marwadi University**.

The experiments focus on core concepts of **Machine Learning, Deep Learning, Natural Language Processing (NLP), and Reinforcement Learning**, implemented using Python in visual studio code. Google Colab.

---

## 📌 Objectives

- Understand fundamental concepts of Artificial Intelligence
- Implement Machine Learning algorithms (supervised & unsupervised)
- Work with NLP techniques like TF-IDF, TextRank, and preprocessing
- Build Deep Learning models using CNN
- Learn and implement Recommendation Systems
- Apply Reinforcement Learning techniques
- Solve real-world AI problems through practical experiments
